# Web-highlighter-extension

Браузерное расширение веб-выделитель, который позволяет сохранять весь выделенный текст вместе с его ссылкой и датой и хранить в локальной памяти браузера. Через popup-окошки расширения можно управлять сохраненным текстом

## Что работает

- Расширение видно в панели расширений браузера- 
- При выделении текста появляется кнопка "сохранить"
- После сохранения текст маркируется желтым цветом
- Текст, URL заголовка и дата создания сохраняются в chrome.storage.local
- Сохраненные данные доступны через popup-окошко, которое можно запустить, нажав на иконку расширения в панели расширений

## Что я узнал во время работы

Как создаются расширения для браузеров и в чем их архитектурное различие от обыкновенных веб-прилжений. 
1) surroundContents - берет диапозон элементов
2) IIFE (Immediately Invoked Function Expression) - функция, которая выполняется мгновенно после ее определения. Примеры:
`( function() {}) ()`
`( () => {}) ()` . Конкретно в этом репозитории она была применена для того, чтобы сохранить внутри себя множество переменных и не засорять глобальную область видимости
3) mouseenter - аналог css псевдокласса :hover
4) window.getSelection(); - встроенный метод получения выделенного пользователем текста
5) getBoundingClientRect - координаты диапазона относительно видимой части экрана
